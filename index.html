4 Manual Server Setup - For advanced users

Run this on your VPS to generate real pairing codes:

node generate-session.js

This connects to WhatsApp servers and generates working 8-digit pairing codes.

Steps:

  1. Clone the repository to your server
  2. Run yarn install
  3. Run node generate-session.js
  4. Enter your phone number when prompted
  5. Use the generated 8-digit code in WhatsApp

  • Real pairing codes only work from servers - they must be generated by WhatsApp/Baileys
